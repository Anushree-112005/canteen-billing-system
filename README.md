An **Online Canteen Billing System** developed using **HTML, CSS, and JavaScript**. The application enables users to register, log in, browse food items, place orders, manage their cart, and generate bills. The system uses **Browser Local Storage** to save user and order information without requiring a backend or database.

# 📌 Overview

The Online Canteen Billing System is designed to simplify food ordering and billing within a college or office canteen. Users can create an account, browse the menu, add food items to their cart, and generate bills instantly. Administrators can monitor customer orders and manage the billing system.

# 💾 Data Storage

The application stores data using **Browser Local Storage**.

Stored Information includes:

- User Details
- Login Credentials
- Cart Items
- Order History
- Billing Details

No backend server or database is required.
